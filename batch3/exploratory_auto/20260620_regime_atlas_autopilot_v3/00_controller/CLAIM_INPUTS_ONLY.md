# CLAIM INPUTS ONLY

This program produces VALIDATED INPUTS for PI claims — ledgers, parquet atlases, factual summaries,
neutral figures, auditable examples. It does NOT write thesis conclusions.
Forbidden phrasings: "this proves the decomposition", "confirms conditioning-bound basins",
"challenges noise-search". Allowed: counts, p_hat+CI, intervention Δp, audit-pending, regime labels
with eligibility flags, and clearly-labeled hypotheses (SURPRISES_AND_LEADS.md only).
Headline-eligibility filter (§12) decides what may reach a claim; exploratory rows stay exploratory.
