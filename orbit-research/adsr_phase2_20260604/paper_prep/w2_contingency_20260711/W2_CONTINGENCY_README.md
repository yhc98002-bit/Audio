# W2 Corrected-Instrument Contingency

`W2_ACTIVATION_STATUS = SCAFFOLDED_NOT_AUTHORIZED`

This harness is activated only if decisive Test 1 returns `demucs_missing`.
The current 50-clip run is a plumbing check and neither relabels frozen
evidence nor changes a Section 7 or `PLAN.md` status.

## Components

- `build_w2_relabel_manifest.py` inventories the 4,096-candidate spine,
  retained batch3/atlas keeps, and the frozen Stage 3 and N2 FLACs.
- `w2_instruments.py` exposes the current Demucs detector, a Demucs+PANNs
  ensemble, a human-calibrated Demucs threshold, and a held-out-validated
  judge adapter.
- `run_w2_relabel.py` writes append-only per-clip results and fails each row
  whose media or instrument output is unavailable.
- `recompute_w2_headlines.py` produces old-versus-corrected clean-rate and N2
  regime diffs. It never edits `PLAN.md`; adoption requires explicit review.

## Fail-Closed Inputs

The Demucs+PANNs route requires a complete audio-hash/PANNs score table and an
explicit `or` or `and` decision rule. The human-threshold route requires a
calibration artifact. The judge route requires strict
`judge:<model>:validated:<gold_set_hash>` provenance, PASS metadata,
calibration metrics, and a hash-verified raw-response ledger.

The inventory found only one surviving original media target for the
4,096-candidate spine; 4,095 historical entries are dangling links and are
recorded as `not_retained_on_disk`. Stage 3 (6,144), N2 (16,384), and atlas
keep (194) media are present. Missing spine media are never regenerated by
this harness.
